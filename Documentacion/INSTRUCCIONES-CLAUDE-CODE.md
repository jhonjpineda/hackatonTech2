- **Backend:** NestJS + TypeScript + Prisma + PostgreSQL
- **Frontend:** Next.js 14 + React 18 + TypeScript + Tailwind CSS
- **Base de Datos:** PostgreSQL 15
- **Cache:** Redis 7
- **WebSockets:** Socket.io
